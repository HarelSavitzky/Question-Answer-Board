Program Explanation

in this program we will build a servlet project, we will build a site that will hold questions and
the user would be able to reply on them and to display the answers if he wish.

Design Explanation

the program designed in a modular way, each servlet/function got its own responsibility,
*questionServlet - the main servlet, initializing the runTimeData (class that function as database), and extracting
the questions and number of answers to each answer in order to return it to the script file that builds the
questions on that page dynamically.
*answerServlet - this servlet is being called if we press on "answer" button, or if we want to submit the answer
if we press the answer button the page will be build dynamically and we will move there, if we press submit
we will extract the name and the answer, we will save them in the database class, than we will be redirected to the
index page. *getAnswers - a servlet that is being called when we press on "show answers", than the servlet extract the data
(questions + answers) and return in to the index script page in order to build the answers display dynamically.
*runTimeData - a class that functions as our database, it saves data on the program runtime only, it has several
functions in order to manage the data. we will create a single object from this class, that will be used from all
the servlets on the program (in the exact same one..).
*index.html - our landing page.
*readme.html - our readme page.
*formJs.js - the form script file.
*indexJs.js - our landing page (question page) script file.
*questions.txt - the questions file.
*styles.css - CSS file.
